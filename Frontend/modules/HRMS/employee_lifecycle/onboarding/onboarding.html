Paste the regular spreadsheet URL or the published CSV link from the "Publish to web" dialog — either works.
📋 How to set this up (one time only)
1
Open your Google Form responses sheet (the Sheet linked to your form)
2
Click File → Share → Publish to web
3
In the popup, select Sheet1 (or your responses tab) and choose Comma-separated values (.csv), then click Publish and confirm
4
Go back to the address bar, copy the full spreadsheet URL (the regular edit URL is fine — e.g. docs.google.com/spreadsheets/d/…) and paste it above
✓
Click Save & Sync Now — candidates will appear automatically and data will refresh on every page load
⚠️
Important: You must use Publish to web (not just "Share → anyone with link"). Only published sheets can be read by the server without requiring a Google login.
